2. 如在微信中打开 → 点右上角 **...** → **在浏览器中打开**
3. 安装时允许 **"来自未知来源的应用"**
4. 打开APP即可使用（完全离线可用）

### Install Steps
1. Scan QR → Open download page → Tap **app-debug.apk** to download
2. If in WeChat → tap **...** (top right) → **Open in Browser**
3. Allow **"Install from unknown sources"** during install
4. Open APP — works fully offline!
